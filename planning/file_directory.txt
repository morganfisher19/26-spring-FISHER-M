File structure:
Project_Root/
├─ code/
│  ├─ data/
│  │  ├─ bronze/
│  │  │  └─ Contains raw data files that are not JSON
│  │  ├─ silver/
│  │  │  └─ Contains raw JSON files
│  │  ├─ gold/
│  │     └─ Contains clean data

Pipeline:

run_pipeline.py          # entry point — runs all steps
config.py                # constants, API keys, paths
pipeline/
  extract/
    congress_api.py      # bills, members, amendments, house rollcall
    house_votes.py       # house XML scraper
    senate_votes.py      # senate XML scraper
  transform/
    clean_members.py
    clean_bills.py
    clean_votes.py
    clean_vote_records.py
  utils/
    cache.py             # cache logic
    api_client.py        # rate-limited session

│  ├─ notebooks/
│  │  └─jupyter notebooks for getting/cleaning data
│  ├─ scripts/
│  │  └─ python scripts for getting/cleaning data
│  ├─ sql/
│  │     └─sql scripts for postgresSQL database
│  ├─ src/
│  └─ website/
│     └─ All files for creating web application (including Flask & React scripts)
