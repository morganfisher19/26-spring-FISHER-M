Data:
- Voting data:
    House voting data - individual votes: 
        Website: https://clerk.house.gov/Votes
        XML data: https://clerk.house.gov/evs/{YEAR}/roll{XXX}.xml
    House voting data - roll call:
        Website: https://api.congress.gov/
        API: https://api.congress.gov/v3/house-vote/119?format=json&offset=0&api_key=
    Senate voting data - individual votes:
        Website: https://www.senate.gov/legislative/votes_new.htm
        XML data: https://www.senate.gov/legislative/LIS/roll_call_votes/vote{CONGRESS}{SESSION}/vote_{CONGRESS}_{SESSION}_{XXXXX}.xml
    Senate voting data - roll call:
        Website: https://www.senate.gov/legislative/votes_new.htm
        XML data: https://www.senate.gov/legislative/LIS/roll_call_lists/vote_menu_119_{SESSION}.xml
- Member data
    Member data:
        Website: https://api.congress.gov/
        API: https://api.congress.gov/v3/member/congress/119?format=json&currentMember=true&api_key=
    Member birth year data:
        Website: https://bioguide.congress.gov/search?index=%22bioguideprofiles%22&size=96&matches=[]&filters={%22jobPositions.congressAffiliation.congress.name%22:[%22The%20119th%20United%20States%20Congress%22]}&sort=[{%22_score%22:true},{%22field%22:%22unaccentedFamilyName%22,%22order%22:%22asc%22},{%22field%22:%22unaccentedGivenName%22,%22order%22:%22asc%22},{%22field%22:%22unaccentedMiddleName%22,%22order%22:%22asc%22}]&view=%22Table%22
        *Downloaded csv locally
    Member bioguides:
        Website: https://bioguide.congress.gov
        Page link:https://bioguide.congress.gov/search/bio/{BIO_ID}
- Bill data
    Laws data:
        Website: https://api.congress.gov/
        API: https://api.congress.gov/v3/law/119?format=json&api_key=
    Bill data:
        Website: https://api.congress.gov/
        API Search: https://api.congress.gov/v3/bill/119/hr?format=json&offset=0&api_key=
        ^ bill type want to include hr, s, hjres, & sjres
        ^ many data points (~8000 alone for those with type hr)
    Amendment data:
        Website: https://api.congress.gov/
        API Search: https://api.congress.gov/v3/amendment/119?format=json&offset=0&api_key=
        ^ with around 5000 data points for 119th congress
- State reference data:
    https://github.com/jasonong/List-of-US-States/blob/master/states.csv
